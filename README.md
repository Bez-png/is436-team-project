## Project Objective
The goal of this project is to practice using GitHub for team collaboration. Each team member will create their own HTML file and push it to the shared repository. This helps us learn version control, commits, and working together on the same project.

## Repository Structure
Each team member will create their own HTML file using their name.

Example:
- abenezer_getachew.html
- john_doe.html
- jane_smith.html

## How to Contribute
1. Pull the latest changes from the repository
2. Create or edit your HTML file
3. Commit your changes
4. Push your changes to GitHub
